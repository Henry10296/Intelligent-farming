## 🚀 使用指南

### 快速打包流程

1. **自动构建** (推荐)
   ```cmd
   cd SmartFarmPackage\scripts
   build.bat
   ```

2. **高级部署**
   ```powershell
   cd SmartFarmPackage\scripts
   .\deploy.ps1 -CreateInstaller -RunTests
   ```

3. **发布准备**
   ```powershell
   .\deploy.ps1 -Version "2.0.1" -PublishToGitHub
   ```

### 项目构建输出

构建完成后将生成:
```
Build/
├── Package/                    # 完整发布包
│   ├── bin/                   # 可执行文件
│   ├── docs/                  # 文档
│   ├── config/                # 配置文件
│   ├── assets/                # 资源文件
│   └── SmartFarmSystem.bat    # 启动脚本
│
├── SmartFarmSystem-v2.0.0.zip # ZIP发布包
└── SmartFarmSystem-Setup.exe  # 安装程序 (可选)
```
